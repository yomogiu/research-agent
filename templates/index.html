How it works

  1. Run the Flask server: python app.py.
  2. In Ghostty, start a session that tails {{ input_path }} for new commands and streams output into {{ output_path }}.
  3. Interact with your CLI AI agent normally. The output pane refreshes every second.

The scripts folder contains an example bridge for macOS that sends commands into an interactive process and syncs output back to the web UI.
